CBR: Case-Based-Reasoning for Survival Data

The R package case-based-reasoning provides an R interface case based reasoning on survival data.

This package can be installed from github by:

install.packages("devtools")
devtools::install_github("sipemu/Similarity")
devtools::install_github("sipemu/case-based-reasoning")

The Similarity-package contains the C++-distance calculation functions for this package. The compilation should work on Ubuntu and Mac OS X out-of-the-box. On Windows you need g++ verion 4.9.3 or greater (Windows- Binary: Similarity_01.zip).

Please use Github issues if you want to file bug reports or feature requests.

1 Features

This R package provides two methods case based reasoning for survival data:

  • Cox-Regression

  • Random Survival Forest (Proximity and Depth Measure)

Besides the functionality of searching similar cases, some additional features are included:

  • automatic validation of the key variables between the query and similar cases dataset

  • checking proportional hazard assumption for the Cox Model

  • C++-functions for distance calculation

2 Near future implementations:

  • analysis of missing values and imputation for the Cox model

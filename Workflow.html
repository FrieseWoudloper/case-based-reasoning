Workflow

Note: Package is in active development. Structure, workflow, and syntax may change dramatically.

Following diagramm describes the workflow of the case-based reasoning R package. Box colour has different meanings:

  • Useable: this features can be used
  • Not useable: planned features
  • Optional: this function will be called by subsequent features and can be called optionally; this means, if you call e.g. call ‘Calculate Distance Matrix’ automatically the node ‘Modell Fitting’ will be called (if not already done!)
